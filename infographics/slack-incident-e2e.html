Key Design Decisions
🔁
ReAct loop — exactly 2 iterations
Tool call 1: get context. Tool call 2: post card. end_turn on success. MAX_ITERATIONS = 10 safety cap prevents runaway loops.
🔌
⚡
Socket Mode — no public URL needed
Bot opens a persistent WebSocket to Slack. Slack pushes events over it. No ngrok, no deployment required for local dev.
📝
Idempotent in-place updates
card is edited via chat_update using the original message ts. Buttons disappear after first click — no double-acknowledgement possible.
💉
Slack client injected at call time
tools.py has no module-level Slack import. Client is passed as a kwarg. Makes all tools fully testable without credentials (dry-run path).
