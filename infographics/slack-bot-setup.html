Day 5 — Slack Incident Bot

Slack App — Setup & Configuration

8 steps from api.slack.com to a running Socket Mode bot on your machine.
Complete steps 1–8 before running python bot.py --trigger ALERT-001.

Steps: 8
Tokens collected: 3
Mode: Socket Mode
Time: ~10 min
App Creation
1
Create a free Slack workspace
▸ slack.com/get-started
Go to slack.com/get-started → Create a Workspace. Name it anything — e.g. aiops-lab. Skip the "invite teammates" screen. You can be the sole member.
2
Create a Slack App from scratch
▸ api.slack.com/apps
api.slack.com/apps → Create New App → From scratch
App name: AIOps Incident Bot · pick your aiops-lab workspace → Create App.
Permissions & Socket Mode
3
Enable Socket Mode → generate App Token
▸ Settings → Socket Mode
Left sidebar → Settings → Socket Mode → toggle ON.
Generate App-Level Token · name: local-dev · scope: connections:write → Generate.
SLACK_APP_TOKEN = xapp-1-…
4
Add Bot Token Scopes
▸ Features → OAuth & Permissions
Scroll to Bot Token Scopes → Add an OAuth Scope → add all four:
chat:write
chat:write.public
channels:read
reactions:write
Events & Interactivity
5
Enable Interactivity (button callbacks)
▸ Features → Interactivity & Shortcuts
Toggle Enable Interactivity → ON.
The Request URL field is required but bypassed by Socket Mode — enter https://placeholder.example.com to pass validation.
6
Subscribe to Bot Events
▸ Features → Event Subscriptions
Enable Events → ON · scroll to Subscribe to bot events → Add Bot User Event:
message.channels
app_mention
Install & Run
7
Install to workspace → get Bot Token
▸ Features → OAuth & Permissions
Click Install to Workspace → Allow.
Copy the Bot User OAuth Token at the top of the page.
SLACK_BOT_TOKEN = xoxb-…
8
Create #incidents channel, invite bot, copy ID
▸ Slack desktop app
Create channel #incidents → channel details → Integrations → Add apps → AIOps Incident Bot.
Copy Channel ID from the URL bar or Details panel (starts with C).
SLACK_CHANNEL_ID = C0XXXXXXXXX
agents/slack-incident-bot/
# 1. copy and fill the env file
cp .env.example .env
 
# 2. install deps into the agent venv
uv venv .venv && uv pip install -r requirements.txt
 
# 3. fire ALERT-001 and keep listening for button clicks
python bot.py --trigger ALERT-001
